
# Lisbon Chair Shop

A simple e-commerce website for a fictional chair shop called "Lisbon Chair Shop." This project showcases a minimalist design with HTML and CSS, providing users with a clean and visually appealing interface to browse chairs.

## Features

- **Home Page**: A landing page with a hero image and a featured section displaying available chairs.
- **Responsive Design**: The website is designed to be responsive and user-friendly on all devices.
- **Images**: Various high-quality images of the chairs are provided to enhance the user experience.

## Project Structure

- `index.html`: The main HTML file containing the structure of the webpage.
- `style.css`: The CSS file that styles the website and ensures responsiveness.
- `hero.jpg`: The hero image displayed on the home page.
- `chair-1.jpg`, `chair-2.jpg`, `chair-3.jpg`: Images of the featured chairs.
- `customers.jpg`: An image of customers, adding a social element to the site.
- `.git`: Git version control folder, containing project history.

## Installation and Usage

To run this project locally, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/krishnakamalbaishnab/lisbon-chair-shop.git
    ```

2. Navigate to the project directory:

    ```bash
    cd lisbon-chair-shop
    ```

3. Open `index.html` in your web browser:

    ```bash
    open index.html
    ```
